# iDEV Vegetable Shop — Frontend

Dự án website bán rau củ sạch online, xây dựng bằng Vue 3 + Vite + Pinia + TailwindCSS.

## Công nghệ sử dụng
- **Vue 3** (Composition API)
- **Vite** (Build tool)
- **Pinia** (State management)
- **Vue Router 4** (Routing)
- **Axios** (HTTP client)
- **Tailwind CSS v4** (Styling)

## Tính năng chính
- Xem danh sách và chi tiết sản phẩm
- Giỏ hàng (offline localStorage + online Database)
- Đăng ký / Đăng nhập với JWT
- Quên mật khẩu qua OTP Email
- Thanh toán COD và VNPay

## Khởi động dự án
```bash
npm install
npm run dev
```
